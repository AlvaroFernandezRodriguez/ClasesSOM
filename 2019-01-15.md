## Get full help on the net user command:
```PowerShell
net help user
```

## Enable/disable system users via command line:
```PowerShell
net user riyaz /active:yes (no)
```

## Run a command as another user:
```MS-DOS
runas /noprofile /user:domain\username "mmc wf.msc"
```
