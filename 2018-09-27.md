# Ejercicios sobre ficheros

## Seleccionar nombre de ficheros
```PowerShell
Get-ChildItem | Select-Object name
```
## Seleccionar todas las propiedades
```PowerShell
Get-ChildItem | Select-Object *
```
## Agrupar
```PowerShell
New-Item nombre
New-Item nombre.1
Get-ChildItem | Group-Object Name
```
## Agrupar por extensi√≥n
```PowerShell
New-Item nombre1.1
New-Item nombre.1

```
